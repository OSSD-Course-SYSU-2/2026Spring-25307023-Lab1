# Git 开发者生态演示项目

## 关于本项目
本项目演示 Git 版本控制的核心功能和开发者生态系统。

## 功能特性
- 版本控制 (git add, commit, log)
- 分支管理 (git branch, checkout, merge)
- 标签管理 (git tag)
- 协作开发 (git remote, push, pull)

## 快速开始
```bash
git clone <repository-url>
cd task2
python hello-world.py
```

## 分支说明
- `main` - 主分支，稳定版本
- `dev` - 开发分支
- `feature/add-greeting` - 功能分支示例
